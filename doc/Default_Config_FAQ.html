Answers

Question: Why am I not following the Marked target?

Answer: You can set the [FollowMarkTarget] option to true to automatically target the MA's Marked target.

Settings Used: FollowMarkTarget

Question: Why am I not announcing the target to the group?

Answer: You can set the [AnnounceTargetGroup] option to true to announce the target over /gsay.

Settings Used: AnnounceTargetGroup

Question: Why am I buffing the people before they are rezzed?

Answer: You can set the [BuffRezables] option to false to skip buffing the people who have corpses nearby.

Settings Used: BuffRezables

Question: How do I change when I maditate?

Answer: You can set the [DoMed] option to the desired Meditation Setting.'Out of Combat' Will only Meditate Outside of Combat.'In Combat' Will Meditate in and out of Combat.'Off' Will not Meditate.

Settings Used: DoMed

Question: Why am I not announcing Heal casts to the group?

Answer: You can set the [HealAnnounceGroup] option to true to announce Heal casts in group.

Settings Used: HealAnnounceGroup

Question: Why are my healers attacking the mob and not healing?

Answer: You can set the [PriorityHealing] option to true to enforce healing over engaging in combat actions.

Settings Used: PriorityHealing

Question: Why am I not announcing Cure casts?

Answer: You can set the [CureAnnounce] option to true to announce Cure casts.

Settings Used: CureAnnounce

Question: Why am I not casting Dots?

Answer: The target may be to low health, Adjust [MobLowHP] option to the minimum health of the target to stop casting dots at.

Settings Used: MobLowHP

Answer: You can set the [ManaToDot] option to the minimum mana to casting dots at.

Settings Used: ManaToDot

Question: I need to heal but I am invisible, how do I fix this?

Answer: You can set the [BreakInvis] option to true to break invis to heal when out of combat only.

Settings Used: BreakInvis

Question: Why am I not meditating?

Answer: Make sure [DoMed] is not set to 'Off'.You can set the [ManaMedPct] option to the percent mana you would like to start meditating at.

Settings Used: ManaMedPct

Question: Why am I not casting Debuffs?

Answer: You can set the [ManaToDebuff] option to the minimum mana to casting debuffs at.

Settings Used: ManaToDebuff

Question: Why am I not healing?

Answer: You can set the [MainHealPoint] option to the percent health you would like to start healing at.

Settings Used: MainHealPoint

Answer: You can set the [MaxHealPoint] option to the percent health you would like to start healing at.

Settings Used: MaxHealPoint

Question: Why am I always out of endurance?

Answer: Make sure [DoMed] is not set to 'Off'.You can set the [EndMedPct] option to the percent endurance you would like to start regenerating at.

Settings Used: EndMedPct

Question: Why am I not healing with my BIG HEAL?

Answer: You can set the [BigHealPoint] option to the percent health you would like to start using the Big Heal Rotation at.

Settings Used: BigHealPoint

Question: Why do I keep dotting the named when it is below [HPStokDot]?

Answer: Named Targets have their own setting for Stop Dot HP, Adjust [NamedLowHP] option to the minimum health of the Named to stop casting dots at.

Settings Used: NamedLowHP

Question: Why am I only Meditating to 60% Mana?

Answer: You can set the [ManaMedPctStop] option to the percent mana you would like to stop meditating at.

Settings Used: ManaMedPctStop

Question: Why am I getting spell not ready spam?

Answer: Your spell may not be ready set the [CastReadyDelayFact] option to the number of milliseconds to wait before checking again.

Settings Used: CastReadyDelayFact

Question: Why am I only Recovering to 60% Endurance?

Answer: You can set the [EndMedPctStop] option to the percent endurance you would like to stop meditating at.

Settings Used: EndMedPctStop

Question: Why are my songs not landing before casting a new one?

Answer: You can set the [SongClipDelayFact] option to the number of milliseconds to wait before casting a new song.

Settings Used: SongClipDelayFact

Question: Why am I only Regenerating to 60% Health?

Answer: You can set the [HPMedPctStop] option to the percent health you would like to stop meditating at.

Settings Used: HPMedPctStop

Settings Used: DoPetHeals

Settings Used: PetHealPoint

Answers

Question: I want to be able to weave in AA nukes when they are available, is this possible?

Answer: Yes, you can enable [WeaveAANukes] to weave in AA Nukes when they are available.

Settings Used: WeaveAANukes

Question: Can I use Mana Burn?

Answer: Yes, you can enable [DoManaBurn] to use Mana Burn when it is available.

Settings Used: DoManaBurn

Question: How do I use my chest item Clicky?

Answer: Enable [DoChestClick] to use your chest item clicky.

Settings Used: DoChestClick

Question: What do the different Modes Do?

Answer: Combo mode is a mix of all 3 types of Nukes, Fire mode will focus only on Fire Nukes, Ice mode will focus only on Ice Nukes, Magic mode will focus only on Magic Nukes.

Settings Used: Mode

Question: How do I use Gift of Mana?

Answer: Enable [DoGOMCheck] to check if you have Gift of Mana before casting big nukes.

Settings Used: DoGOMCheck

Question: How do I use Rain Spells?

Answer: Set the [RainMinHaters] to the minimum number of haters you want to be near the target to use Rain Spells. And make sure you have [DoRain] enabled.

Settings Used: RainMinHaters

Answer: Set the [RainDist] to the minimum distance you want to be from the target to use Rain Spells. And make sure you have [DoRain] enabled.

Settings Used: RainDist

Answer: Set the [RainMinTargetHP] to the minimum HP % you want the target to be at before using Rain Spells. And make sure you have [DoRain] enabled.

Settings Used: RainMinTargetHP

Answer: Enable [DoRain] to use Rain Spells when they are available.

Settings Used: DoRain

Question: Can I customize when to use Jolt?

Answer: Yes, you can set the aggro % at which to use Jolt with the [JoltAggro] setting.

Settings Used: JoltAggro

Question: How do I use Snare Spells?

Answer: Enable [DoSnare] to use Snare Spells when they are available.

Settings Used: DoSnare

Question: How do I use Harvest Spells?

Answer: Set the [HarvestManaPct] to the minimum mana % you want to be at before using a harvest spell or aa.

Settings Used: HarvestManaPct

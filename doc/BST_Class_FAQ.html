Answers

Question: How do I use my AoE Abilities?

Answer: Make sure you have [DoAoe] enabled.

Settings Used: DoAoe

Question: How do I use Feralgia instead of the Growl Spell?

Answer: Enable [DoFeralgia] to use Feralgia for the Growl Effect on your Pet instead of the Growl Spell. This will save you mana and a spell slot.

Settings Used: DoFeralgia

Question: How do I use my Swarm Pet?

Answer: Enable [DoSwarmPet] to use your Swarm Pet spell in addition to Feralgia. This will save you mana and a spell slot.

Settings Used: DoSwarmPet

Question: How do I make my pet the tank?

Answer: Enable [DoTankPet] to use abilities designed for your pet to tank. Disable [DoTankPet] to use abilities designed for your pet to DPS.

Settings Used: DoTankPet

Question: How do I use my Chest item Clicky?

Answer: Make sure you have [DoChestClick] enabled. This will use your chest item clicky during burns.

Settings Used: DoChestClick

Question: My Pet Keeps Dying, What Can I Do?

Answer: Make sure you have [DoPetHeals] enabled. If your pet is still dying, consider using [PetHealPct] to adjust the pet heal threshold.

Settings Used: DoPetHeals

Question: I want to help with healing, what can I do?

Answer: Make sure you have [DoHeals] enabled. If you want to help with pet healing, enable [DoPetHeals].

Settings Used: DoHeals

Question: How do I use my Feign Death?

Answer: Make sure you have [AggroFeign] enabled. This will use your Feign Death AA when you have aggro at low health or aggro on a RGMercsNamed/SpawnMaster mob.

Settings Used: AggroFeign

Question: How do I use my Avatar Buffs?

Answer: Make sure you have [DoAvatar] enabled. Also double check [DoBuffs] is enabled so you can cast on others.

Settings Used: DoAvatar

Question: How do I use my Pet Slow Proc?

Answer: Enable [DoPetSlow] to use your Pet Slow Proc Buff. This does not stack with Pet Damage or Snare Proc Buff.

Settings Used: DoPetSlow

Question: How do I use my Run Speed Buffs?

Answer: Make sure you have [DoRunSpeed] enabled. Also double check [DoBuffs] is enabled so you can cast on others.

Settings Used: DoRunSpeed

Question: How do I use my Emergency Mitigation Abilities?

Answer: Make sure you have [EmergencyStart] set to the HP % before we begin to use emergency mitigation abilities.

Settings Used: EmergencyStart

Question: What is the difference between the modes?

Answer: Beastlords currently only have one Mode. This may change in the future.

Settings Used: Mode

Question: How do I use my Paragon Spirits Abilities?

Answer: Make sure you have [DoParagon] enabled. Set the [ParaPct] to the minimum mana % before we use Paragon of Spirit. Set the [FParaPct] to the minimum mana % before we use Focused Paragon. If you want to use Focused Paragon outside of combat, enable [DowntimeFP].

Settings Used: DoParagon

Question: How do I use my Pet Snare Proc?

Answer: Enable [DoPetSnare] to use your Pet Snare Proc Buff. This does not stack with Pet Damage or Slow Proc Buff.

Settings Used: DoPetSnare

Question: How do I use my Slow Spell?

Answer: Make sure you have [DoSlow] enabled. If you have a Slow AA, make sure it is enabled in your AA Loadout.

Settings Used: DoSlow

Question: How do I use my Damage Over Time Spells?

Answer: Make sure you have [DoDot] enabled.

Settings Used: DoDot

Question: How do I use my Epic Weapon?

Answer: Enable [DoEpic] to click your Epic Weapon.

Settings Used: DoEpic

Question: Why am I not using my Paragon Abilities?

Answer: Make sure you have [DoParagon] enabled. Set the [ParaPct] to the minimum mana % before we use Paragon of Spirit. Set the [FParaPct] to the minimum mana % before we use Focused Paragon. If you want to use Focused Paragon outside of combat, enable [DowntimeFP].

Settings Used: ParaPct

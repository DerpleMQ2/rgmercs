Answers

Question: I want to use my Unity AA, how do I do that?

Answer: Set the [DoUnity] setting to true and the Necro will use Mortifiers Unity when it is available.

Settings Used: DoUnity

Question: I want to stop Liching at a certain Mana %, how do I do that?

Answer: Set the [StopLichMana] setting to the desired % of Mana you want to stop Liching at.

Settings Used: StopLichMana

Question: I want to only use a Rogue Pet for the Backstabs, how do I do that?

Answer: Set the [PetType] setting to Rog and the Necro will only summon Rogue pets.

Settings Used: PetType

Question: I am using Death Bloom to early or late, how do I adjust it?

Answer: Set the [DeathBloomPercent] setting to the desired % of mana you want to cast Death Bloom at.

Settings Used: DeathBloomPercent

Question: I want to use my Battle Rez, how do I do that?

Answer: Set the [BattleRez] setting to true and the Necro will use their Battle Rez during combat.

Settings Used: BattleRez

Question: I want to use Wake the Dead when I have X corpses nearby, how do I do that?

Answer: Set the [WakeDeadCorpseCnt] setting to the desired number of corpses you want to cast Wake the Dead at.

Settings Used: WakeDeadCorpseCnt

Question: I want to use my Lich spells, how do I do that?

Answer: Set the [DoLich] setting to true and the Necro will use Lich spells. You will also want to set your [StopLichHP] and [StopLichMana] settings to the desired values so you do not Lich to Death.

Settings Used: DoLich

Question: What do the different Modes Do?

Answer: Currently Necros only have one mode, which is DPS. This mode will focus on DPS and some utility.

Settings Used: Mode

Question: I want to use my Life Burn AA, how do I do that?

Answer: Set the [DoLifeBurn] setting to true and the Necro will use Life Burn AA if their aggro is below 25%.

Settings Used: DoLifeBurn

Question: I want to stop Liching at a certain HP %, how do I do that?

Answer: Set the [StopLichHP] setting to the desired % of HP you want to stop Liching at.

Settings Used: StopLichHP

Question: How do I manage my aggro with Feign Death?

Answer: Set the [StopFDPct] setting to the desired % of aggro you want to Stand up from FD at.

Settings Used: StopFDPct

Answer: Set the [StartFDPct] setting to the desired % of aggro you want to FD at.

Settings Used: StartFDPct

Question: I want to use my Snare spells, how do I do that?

Answer: Set the [DoSnare] setting to true and the Necro will use Snare spells.

Settings Used: DoSnare

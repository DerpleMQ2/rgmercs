Answers

Question: I want to make sure my pet is always armed, how do I do that?

Answer: You can use the [DoPetWeapons] feature to summon pet weapons.

Settings Used: DoPetWeapons

Question: I want to use Force of Elements AA in my rotation, how do I do that?

Answer: You can use the [DoForce] feature to use the Force of Elements AA in your rotation.

Settings Used: DoForce

Question: Why am I Not summoning Mod Rods?

Answer: You can adjust the mana percentage to begin summoning Mod Rods in combat by setting the [GroupManaPct] setting. Also Make sure you have the [CombatModRod] setting enabled if you want to resummon them during combat. Finally make sure you have the [SummonModRods] setting enabled.

Settings Used: GroupManaPct

Question: I want to use Magic Nukes instead of Fire Nukes, how do I do that?

Answer: You can use the [DoMagicNuke] feature to use Magic Nukes instead of Fire Nukes.

Settings Used: DoMagicNuke

Question: Can I specify the type of pet I want to use?

Answer: Yes, you can select the type of pet you want to summon using the [PetType] setting.

Settings Used: PetType

Question: Why am I not summoning Mod Rods?

Answer: You can adjust the number of party members that need to be under the above mana percentage to summon Mod Rods in combat by setting the [GroupManaCt] setting. Also Make sure you have the [CombatModRod] setting enabled if you want to resummon them during combat. Finally make sure you have the [SummonModRods] setting enabled.

Settings Used: GroupManaCt

Question: How do I use my chest clicky?

Answer: You can use the [DoChestClick] feature to click your chest item.

Settings Used: DoChestClick

Question: Why do I always have items stuck on the cursor?

Answer: You can adjust the delay before autoinventory by setting the [AISelfDelay] setting. Increase the delay if you notice items left on cursors regularly.

Settings Used: AISelfDelay

Answer: You can adjust the delay before autoinventory by setting the [AIGroupDelay] setting. Increase the delay if you notice items left on cursors regularly.

Settings Used: AIGroupDelay

Question: What is the difference between the modes?

Answer: Fire Mode will use Fire Nukes and strive for DPS. PetTank mode will Focus on keeping the Pet alive as the main tank.

Settings Used: Mode

Question: Can i summon mod rods in combat?

Answer: Yes, you can summon mod rods in combat by setting the [CombatModRod] setting. Otherwise we will only summon them during Downtime.

Settings Used: CombatModRod

Question: Can I specify the mod rod item I want to use?

Answer: Yes, you can specify the mod rod item you want to use by setting the [SelfModRod] setting.

Settings Used: SelfModRod

Question: I want to make sure my pet is always Heirloomed, how do I do that?

Answer: You can use the [DoPetHeirlooms] feature to summon pet Heirlooms.

Settings Used: DoPetHeirlooms

Question: How do I adjust when to use Gather Mana?

Answer: You can adjust when to use Gather Mana by setting the [GatherManaPct] setting.

Settings Used: GatherManaPct

Question: I have suspend Minion AA, how do I keep a spare pet suspended?

Answer: You can use the [DoPocketPet] feature to keep a spare pet suspended.

Settings Used: DoPocketPet

Question: I want to use AE Malo in my rotation, how do I do that?

Answer: You can use the [DoAEMalo] feature to use AE Malo in your rotation.

Settings Used: DoAEMalo

Question: My pet keeps dying, how do I keep it alive?

Answer: You can set the [PetHealPct] to a lower value to heal your pet sooner. Also make sure that [DoPetHeals] is enabled.

Settings Used: PetHealPct

Question: I want to use Malo in my rotation, how do I do that?

Answer: You can use the [DoMalo] feature to use Malo in your rotation.

Settings Used: DoMalo

Question: I want to make sure my pet is always armored, how do I do that?

Answer: You can use the [DoPetArmor] feature to summon pet armor.

Settings Used: DoPetArmor

Question: Can I summon mod rods for my group?

Answer: Yes, you can summon mod rods for your group by setting the [SummonModRods] setting.

Settings Used: SummonModRods

Answers

Question: I want to use Canni AA, how do I do that?

Answer: Setting [DoAACanni] to enabled will use Canni AA during downtime.

Settings Used: DoAACanni

Question: Why am I not casting AE slow whenthere is a group of mobs in camp?

Answer: Setting [AESlowCount] to 3 will start AE slowing when there are 3 or more mobs in camp. Also make sure that you enabled [DoAESlow]

Settings Used: AESlowCount

Question: I want to use Run Speed, how do I do that?

Answer: Setting [DoRunSpeed] to enabled will use Run Speed Spells/AAs when available.

Settings Used: DoRunSpeed

Question: What do the different Modes Do?

Answer: Healer Mode will focus on Healing and Buffing the Group. Hybrid Mode will focus on DPS and Healing.

Settings Used: Mode

Question: How do I configure my Canni Dance?

Answer: Setting [AACanniManaPct] to 70 will use Canni AA when you are below 70% mana.

Settings Used: AACanniManaPct

Answer: Setting [AACanniMinHP] to 70 will Stop using Canni AA when you are above 70% HP.

Settings Used: AACanniMinHP

Answer: Setting [SpellCanniManaPct] to 70 will use Canni Spell when you are below 70% mana.

Settings Used: SpellCanniManaPct

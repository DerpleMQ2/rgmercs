RGMercs Lua Edition: Commands FAQ

Command Usage Description
chaseon /rgl chaseon Chase your current target or
campoff /rgl campoff Clear your camp
campon /rgl campon Set a camp here
chaseoff /rgl chaseoff Turn Chasing Off
makeammo /rgl makeammo ## Make ## number of Class 3 Wood Silver Tip Arrows. Minimum of 5
setmode /rgl setmode Make sets my mode to mode - this is a helper so you don't need to use an index.
setitem /rgl setitem Set the Item on your Cursor's Normal loot setting (sell, keep, destroy, ignore, quest, tribute, bank).
lootreload /rgl lootreload Reloads your loot settings.
setglobalitem /rgl setglobalitem Set the Item on your Cursor's GlobalItem loot setting (sell, keep, destroy, ignore, quest, tribute, bank).
lootimport /rgl lootimport Imports from INI to LootRules DB. Incase you were running in standalone mode prior to running RGMercs Lua. This may cause a some lag. Only Run on ONE Character to populate / update the DB.
tribute /rgl tribute Tribute gear to targeted Tribute Master.
reportloot /rgl reportloot Open the LootnScoot Loot History Table.
lootui /rgl lootui Toggle the LootnScoot console UI.
buy /rgl buy Restock Buy Items from your Target.
sell /rgl sell Use lootnscoot to sell stuff to your Target.
loot /rgl loot Loot the Mobs around you.
bank /rgl bank Stash stuff in the bank
cleanbags /rgl cleanbags Destroy the Trash marked as Destroy in your Bags.
pulldenyrm /rgl pulldenyrm "" Removes from the pull deny list
pulldeny /rgl pulldeny "" Puts on the pull deny list
pullallowrm /rgl pullallowrm "" Removes from the pull allow list
pullallow /rgl pullallow "" Puts on the pull allow list
pulltarget /rgl pulltarget Pulls your current target using your rgmercs pull ability
exportwiki /rgl exportwiki Export the FAQ to Wiki Files by Module.
